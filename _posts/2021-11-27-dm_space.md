Using magic Photoshop tools, I was able to create this interplanetary diorama. The starts in the background were created using the noise tool, creating noise and reshaping portions of that noise to create differently spaced and sized stars. The nebulas were created using an underlying gradient from the top left to the bottom right of the screen, as you may be able to tell by the color transformation from there to there. Using that gradient layer, you are able to paint over the background these cloud-like patterns that form the nebula.
